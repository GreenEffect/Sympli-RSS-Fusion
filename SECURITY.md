# Security Policy

## EN

### Supported versions

The main branch is maintained as a priority.

### Reporting a vulnerability

Please do not open a public issue for security reports.

Contact: contact@green-effect.fr

Please include:

- Vulnerability description
- Reproduction steps
- Estimated impact
- Affected version or commit

## FR

### Versions supportées

La branche principale est maintenue en priorité.

### Signaler une vulnérabilité

Merci de ne pas ouvrir d'issue publique pour un signalement sécurité.

Contact: contact@green-effect.fr

Merci d'inclure:

- Description de la vulnérabilité
- Étapes de reproduction
- Impact estimé
- Version ou commit concerné
